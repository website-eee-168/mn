1. set up or check file in git use : git init 2. check file new or edit use : git status 3. Add file code to git use : 3.1. git add namefile example : git add index.html 3.2. add code edit all to git use : git add . 4.Add & Check commit 4.1.add subject(commit) for file write code to git : git commit -m "" 4.2.check subject(commit) for file code in git use : git log 5.Back code(project) to first or second or third use : git checkout commit(code) example : git checkout a84d26e0dff30ab243f4687554b5f2f1d24debe1 6.Back code to The end in git use : git checkout name(branch) 7.Check name branch use : git branch 8. Create & Change branch 8.1 Creat new branch in git use : git branch name(branch) 8.2 And change branch use : git checkout name(branch)
